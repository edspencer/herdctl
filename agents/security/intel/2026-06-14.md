## New Vulnerabilities Discovered

### 1. esbuild RCE via NPM Registry Manipulation (NEW)
- **Advisory ID:** GHSA-gv7w-rqvm-qjhr (1120679)
- **Severity:** HIGH
- **Package:** esbuild
- **Vulnerable Versions:** >=0.17.0 <0.28.1
- **Fixed Version:** >=0.28.1
- **Attack Vector:** Remote code execution by manipulating NPM_CONFIG_REGISTRY environment variable
- **Impact:** Arbitrary code execution during build process
- **Affected:** docs package (Astro dependency chain)
- **Discovery Date:** 2026-06-14
- **Reference:** https://github.com/advisories/GHSA-gv7w-rqvm-qjhr

**Technical Details:**
esbuild's Deno module downloads pre-compiled binaries from npm without integrity verification. An attacker controlling NPM_CONFIG_REGISTRY can redirect binary downloads to a malicious server, achieving remote code execution during the build process.

**Exploitation Scenario:**
1. Attacker sets NPM_CONFIG_REGISTRY to malicious server
2. Developer/CI runs `pnpm build` or `pnpm install`
3. esbuild downloads malicious binary from attacker's registry
4. Malicious code executes with developer/CI privileges

**Mitigation:**
Update esbuild to >=0.28.1 which includes binary integrity verification.
