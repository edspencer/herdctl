---

### ✅ Discord File Attachments - Comprehensive Security Controls (Commit 3f947a0)
**Impact:** NEW ATTACK SURFACE with proper mitigations

**Controls Implemented:**
- ✅ MIME type whitelist (default: images, PDFs, text)
- ✅ File size limits (10 MB default)
- ✅ Max files per message (5 default)
- ✅ Path traversal protection (rejects `..` and absolute paths)
- ✅ UUID-based subdirectories
- ✅ Download timeouts (30 seconds)
- ✅ Automatic cleanup after processing
- ✅ Docker path translation

**Residual Risks:**
- ⚠️ Text files inlined into prompts (potential prompt injection)
- ⚠️ No malware scanning for uploaded files
- ⚠️ Files fetched from Discord CDN without additional verification

**Recommendation:** Consider adding malware scanning for production use (Q15)

---

### ✅ Voice Transcription - Privacy Considerations (Commit 3f947a0)
**Impact:** NEW THIRD-PARTY INTEGRATION

Voice messages transcribed via OpenAI Whisper API.

**Security Controls:**
- ✅ API key from environment (not hardcoded)
- ✅ 60-second timeout on API calls
- ✅ Response validation

**Privacy Concerns:**
- ⚠️ Audio sent to third party (OpenAI)
- ⚠️ No documented data retention policy

**Recommendation:** Document data flow, research OpenAI retention policy (Q16)
